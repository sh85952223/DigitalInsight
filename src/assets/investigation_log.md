## 📂 FILE 01. 수사 착수: 소비 생활의 변화 (Step 0-1)

**Q1. 과거와 비교했을 때, 우리의 '소비(물건을 사는 방식)'는 어떻게 변했나요? 아래 증거 사진(게임 내 오프닝 화면)을 참고하여 작성해 보세요.**
*   (힌트: 과거에는 시장이나 마트에 직접 갔지만, 지금은?)

| 수사 항목 | 과거 (Analog) | 현재 (Digital) |
| :-- | :-- | :-- |
| **장소** | 시장, 백화점, 마트 등 물리적 공간 | ( 예시: 스마트폰 앱, 인터넷 쇼핑몰 ) |
| **결제** | 현금, 지폐 | ( 예시: 간편 결제, 지문 인식, 앱 카드 ) |
| **배송** | 직접 들고 옴 | ( 예시: 새벽 배송, 문 앞 배달 ) |

**💡 수사 결론:** 우리의 일상은 오프라인에서 **( 온라인/디지털 )** 환경으로 급격하게 이동했다.

## 📂 FILE 02. X-RAY 분석: 수상한 설계의 발견 (Step 0-3)
> **현장 기록:** "평범해 보이는 웹사이트... 하지만 특수 고글(X-Ray)을 쓰자 숨겨진 의도들이 붉게 드러났다."

**Q2. 사이트 곳곳에 숨겨져 있던 '사용자를 헷갈리게 만드는 장치'들을 발견했습니다. 각각 어떤 문제가 있었나요?**

1.  **초조한 시계 (Timer):**
    *   *발견:* "마감 임박! 5분 32초 남음!"
    *   *문제점:* ( 예시: 실제로는 시간이 충분한테도, 마음을 조급하게 만들어 충동구매를 유도한다. )
2.  **숨바꼭질 버튼 (Hidden X):**
    *   *발견:* 광고 팝업창의 닫기 버튼이 보이지 않음.
    *   *문제점:* ( 예시: 닫기 버튼을 아주 작게 만들거나 배경색과 비슷하게 숨겨서, 광고를 실수로 누르게 만든다. )
3.  **유령 재고 (Fake Stock):**
    *   *발견:* "현재 2개 남았습니다! (실제로는 재고 많음)"
    *   *문제점:* ( 예시: 물건이 곧 동날 것처럼 거짓말을 하여 당장 사게끔 압박한다. )

**💡 수사 결론:** 사용자의 실수를 유도하거나 심리를 조작하는 나쁜 설계를 **( 다크 패턴 / Dark Pattern )** 이라고 부른다.

## 📂 FILE 03. 안전지대: 엄지손가락의 비밀 (Step 4)
> **현장 기록:** "데이터의 바다를 지나 안전한 곳에 도착했다. 이곳은 나의 엄지손가락이 가장 편안해하는 구역이다."

**Q3. 스마트폰을 한 손으로 잡았을 때, 엄지손가락이 가장 닿기 편한 곳(Easy Zone)은 어디인가요?**
*   *위치:* ( 예시: 화면의 하단 부분 )
*   *이유:* ( 예시: 손가락을 무리하게 뻗지 않아도 자연스럽게 닿을 수 있기 때문이다. )

**Q4. 만약 자주 쓰는 '뒤로 가기'나 '확인' 버튼이 화면 맨 꼭대기에만 있다면 어떨까요?**
*   *나의 생각:* ( 예시: 손가락이 닿지 않아서 두 손을 써야 하거나, 핸드폰을 고쳐 잡다가 떨어뜨릴 수도 있어 매우 불편할 것이다. )

**💡 수사 결론:** 엄지손가락이 닿기 쉬운 곳(Easy Zone)은 사용하기 **편리**하지만, 반대로 기업이 우리가 누르길 원하는 버튼을 배치하여 선택을 **( 유도 )** 하기에도 가장 좋은 위치다. 편리함 뒤에 숨겨진 '설계의 의도'를 꿰뚫어 볼 수 있어야 한다.

## 📂 FILE 04. 중세 카페: 두 가지 세계의 비밀 (Step 5)
> **현장 기록:** "낡은 카페에서 마주한 진실. 보이는 것이 전부가 아니다."

**Q5. 교관(상점 주인)은 카페를 예시로 두 가지 중요한 개념을 설명했습니다. 각각 무엇을 의미하는지 연결해 보세요.**

| 개념 (Concept) | 카페에서의 예시 (Analogy) | 정의 (Definition) |
| :-- | :-- | :-- |
| **U****** (보이는 것) | **( 예시: 카페의 인테리어, 메뉴판 디자인, 컵의 모양 )** | 사용자가 눈으로 보고 직접 만지는 '도구'나 '화면' |
| **U****** (겪는 것) | **( 예시: 점원의 친절한 인사, 주문 후 커피를 받는 과정, 편안한 분위기 )** | 사용자가 그 도구를 사용하며 느끼는 '총체적인 경험' |

**💡 수사 결론:** 예쁜 인테리어(**UI**)도 중요하지만, 손님이 기분 좋게 커피를 마시는 과정(**UX**)이 엉망이라면 좋은 카페가 아니다.
